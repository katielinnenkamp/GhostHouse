1. Introduction

Thank you for using the Stylized Wooden Bathroom Kit – Low Poly Asset Pack!  
This pack provides a cozy and handcrafted collection of bathroom-themed props, ideal for stylized, mobile, or fantasy environments.


2. Package Contents
-------------------

4. How to Use in Unity
----------------------
1. Import the package into Unity.  
2. Open Scenes/Demo.unity to preview all props in a styled environment.  
3. Drag any prefab from the Prefabs/ folder into your scene.  
4. Materials and textures can be easily swapped from the organized folders inside Art/.  


5. Demo Scene Overview
----------------------
Demo.unity  
- Showcases all bathroom props with lighting setup, camera, and color variants in a cozy interior layout.  
- Useful for previews, integration tests, or promotional screenshots.
